# codePTIT++ - Tiện ích hỗ trợ lập trình CodePTIT

codePTIT++ là tiện ích mở rộng giúp cải thiện trải nghiệm làm bài trên nền tảng CodePTIT. Tiện ích tập trung vào việc giảm thao tác lặp lại và bổ sung những tính năng nhỏ nhưng hữu ích.

Tính năng:
- Trình soạn thảo code trực tiếp: Hỗ trợ C/C++, Java, Python, C# với tô sáng cú pháp và font chữ JetBrains Mono.
- Nút Dán & Nộp: Nạp code trực tiếp từ clipboard mà không cần chọn file.
- Sao chép thông minh: Tự động loại bỏ dòng trống và khoảng trắng thừa khi copy Input/Output mẫu.
- Hỗ trợ Competitive Companion: Tự động điền tiêu đề, giới hạn và ví dụ khi parse bài sang IDE (VS Code, CP Editor...).
- Phím tắt: Ctrl/⌘+Enter để nộp bài, Ctrl/⌘+Shift+V để dán và nộp nhanh, Ctrl/⌘+Shift+X để gửi đề bài sang IDE.

Hướng dẫn cài đặt Competitive Companion:
1. Mở Extension options của Competitive Companion.
2. Tại mục Custom rules, thêm Regex: ^https://code\.ptit\.edu\.vn/(student/question|beta/problems)/.*
3. Chọn CodeforcesProblemParser ở ô bên cạnh.
